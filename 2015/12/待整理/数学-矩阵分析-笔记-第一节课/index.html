矩阵分析笔记-第一节课 背景介绍
2015-12-01| 数学 矩阵分析
Word count: 495|Reading time: 2 min

摘要（Abstract）

本文摘要。

背景知识

线性空间

线性变换 T ：V1 —–> V1

维数 dim V （我们考虑有限维）

矩阵 <—–相互对应—-> 线性变换

同一个线性变换可以对应不同的矩阵（因有不同的basis）

以后以讨论矩阵为主，但是矩阵的背后有线性变换的概念支撑。

线性空间和矩阵的对应

已知：有线性变换T ：V1 —–> V1
有如下变换：
{e1}—T—> {fj} :A
{e`1}—T—> {fj} ：B
则AB矩阵的关系：
A(m*n) = B(m*n) X(n*n)

有如下变换：
{e1}—T—> {fj} :A
{e`1}—T—> {fj} ：B
则AB矩阵的关系：
A(m*n) = X(n*n)B(m*n)

如有以下关系：V1 = V1 dimV = n
则：
A = X-1 B X
我们称之为A,B similar (那我们就知道了，所谓的相似矩阵，实际上就是同一个线性变换 ，但是basis不同而已)

内积与线性空间的对应

已知: 有线性变换V1 V1
对应的basis {ei} {fj}
同上的线性空间和矩阵的对应 则有：
A = B U (Un*n 是 Unitary matrix 满足 U*U =UU* = I n)
A = V B
A = V B U
A = U*B U =UBU*

几个相关概念

Unitary matrix 酉矩阵 ： n阶复方阵U的n个列向量是U空间的一个标准正交基，则U是酉矩阵(Unitary Matrix)。显然酉矩阵是正交矩阵往复数域上的推广。
正交矩阵 ：是实数特殊化的酉矩阵，因此总是正规矩阵。尽管我们在这里只考虑实数矩阵，这个定义可用于其元素来自任何域的矩阵。正交矩阵毕竟是从内积自然引出的，对于复数的矩阵这导致了归一要求。

标准式的概念

为了比较、判断几个矩阵的不同而产生的。所以有以下要求：要简单（好比较）；要所有的矩阵都可以化成标准式；
