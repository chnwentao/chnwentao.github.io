网络编程的三要素

IP地址

点分十进制（一个字节0~255） IP地址的分类：分ABCD共5类

特殊的IP地址： 127.0.0.1 本机地址 x.x.x.255 广播地址 x.x.x.0 网络地址 InetAddress类（略）

端口

是应用程序的标识。范围：0-65535。其中0-1024不建议使用。

协议

UDP:数据打包,有限制,不连接,效率高,不可靠 TCP:建立数据通道,无限制,效率低,可靠

Socket机制

网络上具有唯一标识的IP地址和端口号组合在一起才能构成唯一能识别的标识符套接字。 通信两端都应该有Socket对象、 所有的通信都是通过Socket间的IO进行操作的
