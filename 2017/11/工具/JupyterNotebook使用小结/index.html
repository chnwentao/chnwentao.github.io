为什么要用 Jupyter NoteBook

IPython 是一个 Python REPl shell，环境远比 Python 自带的强大，而 Jupyter Notebook 则是一个基于 IPython REPl 的 Web 应用，运行结果可保存为后缀.ipynb，交互性强，所见即所得，数据分析，写分析报告等的不二利器。

NoteBook 几乎涵括了 QT Console 具有的全部交互式功能，如代码高亮，自动补全，实时帮助，内嵌显示绘图结果等.

其次，计算过程及结果可以方便地保存为多种格式，如默认的 JSON 格式，Python 脚本以及 PDF 等；此外，借助 Markdown 及 MathJax，我们可以在计算过程中插入详尽细致的描述，甚至以描述为主，计算为辅!

对我厂来说,最主要的是可以远程访问!!! 这样多们就可以操作本机一样在本地的浏览器上,互交的操作远程的虚拟机(测试机)啦!

前期工作

在我厂的机子上用任何工具都是一个很艰难的过程:

  • 安装 jumbo : bash -c "$( curl http://jumbo.baidu.com/install_jumbo.sh )"; source ~/.bashrc

  • 安装 python : jumbo install python

  • 安装 setuptools (pip 依赖 setuptools，in python 2.7): jumbo install python-setuptools

  • 安装 pip : jumbo install python-pip

-

  • 安装 ipython : pip install ipython

  • 安装 jupyter : pip install jupyter

到这里 我们就可以愉快的使用 Jupyter NoteBook 啦!
