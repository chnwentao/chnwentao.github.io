scrapy 安装及使用
2016-04-15| 工具
Word count: 479|Reading time: 3 min

安装

下面是我的安装过程，有点坎坷，花了一个下午，，本人环境是OS X EI Capitan

安装pip

pip用来管理python的各种包，类似于Redhat的yum
sudo easy_install pip

安装scrapy

sudo pip install scrapy
报错如下，表明uninstall six出错

安装six 1.10.0

上面的问题是需要安装six 1.10.0，目前机器环境是six 1.4.0，然后卸载six 1.4.0出错

six的官网地址如右：https://pypi.python.org/pypi/six/1.10.0#downloads

下载包直接python setup.py install即可

再次安装scrapy

再次执行sudo pip install scrapy即可安装成功
