序列类型函数sorted(list)

sorted(iterable，cmp，key，reverse） 参数：

  • iterable可以是list或者iterator；
  • cmp是带两个参数的比较函数；
  • key 是带一个参数的函数；
  • reverse为False或者True(代表升序或者降序)

sort & sorted的区别

sorted(list)返回一个对象。原来的list不变，生成一个新的排好序的list对象。 list.sort() 不会返回对象，改变原有的list。 sort函数是list列表中的函数，而sorted可以对list或者iterator进行排序（不是list也可以）
